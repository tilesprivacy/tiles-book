# Tilekit

Monorepo hosting both in-house and vendored libraries and tools powering the Tiles Launcher.

## Quick Start

First, run `pnpm i` to install the dependencies.

Then, run `pnpm dev` to start the development server and visit localhost:3000.

## Documentation

This documentation site provides comprehensive information about the various components, libraries, and tools available in the Tilekit monorepo.
