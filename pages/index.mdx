# Tilekit
 
Tilekit is a declarative, cross-platform [Modelfile](https://ollama.readthedocs.io/en/modelfile/)-based SDK that lets developers customize open models and agent experiences.

## Documentation

This documentation covers the various components, libraries, and tools available in the Tilekit monorepo.
